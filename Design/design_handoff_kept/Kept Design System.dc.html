OfferBee turns premium-card benefits into a calm monthly review. This document defines the visual language: foundations (color, type, spacing, materials) and the component set that builds every screen. Two palettes ship — Honey (light) and Onyx (dark) — over one fixed layout and type system.

01 — Foundations

Color

Tokens are semantic, not literal — components reference roles (surface, accent, warning) so a whole theme swaps by remapping the palette. Card-brand colors sit outside the theme and never change.
